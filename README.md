# Puesta en marcha del Frontend

## Requisitos previos

Antes de ejecutar el frontend, es necesario tener instalado **Node.js** en el equipo. La instalación de Node.js incluye también **npm**, que será utilizado para gestionar las dependencias del proyecto.

## Instalación y ejecución

1. Clonar el repositorio del frontend en la ubicación deseada.
2. Abrir una terminal dentro de la carpeta raíz del proyecto.
3. Instalar las dependencias ejecutando el siguiente comando:

```bash
npm install
```

4. Una vez finalizada la instalación, iniciar la aplicación con:

```bash
npm run dev
```

5. Al ejecutarse el comando, la terminal mostrará la dirección local donde se encuentra disponible la aplicación, por ejemplo:

```text
http://localhost:3000
```

6. Abrir dicha dirección en un navegador web para acceder al sistema.

## Observaciones

Para que el sistema funcione correctamente, es necesario que el backend se encuentre en ejecución antes de utilizar las funcionalidades que requieran comunicación con la API.

Una vez iniciados tanto el frontend como el backend, se podrá acceder a todas las pantallas y funcionalidades disponibles en la aplicación.
